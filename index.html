Buatlah program (Create,Update,Delete) mahasiswa , Pengerjaan Menggunakan PHP dengan Framework Codeigniter dan database Mysql, Berikut Soalnnya.


1.Buat table mahasiswa dengan struktur.

id_mahasiswa int(11) primary_key AI
nama_mahasiswa varchar(200) not null
jurusan varchar(100) not null
kelas varchar(50) not null
foto text not null





2.Menggunakan Template Framework Bootstrap untuk tampilannya dan Form Validasi dari Codeigniter




3.Kemudian buatlah form sederhana untuk menginput, mengupdate dan menghapus sesuai dengan query yang sudah kamu buat pada soal dengan menggunakan fungsi jQuery.




4. Setelah selesai tampilkan data mahasiswa dengan menggunakan table dengan plugin datatables + jquery.
